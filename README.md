# Django + React App 
To do app made using Django Rest Framework and React


## Installation
1. `pip freeze > requirements.txt`
2. `cd toDoApi`
3. `python manage.py runserver`

## React Files
`ToDoApi/to_do_app`

### Note
`/build` folder is necesary
